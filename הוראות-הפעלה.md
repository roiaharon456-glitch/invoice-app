# הוראות הפעלה - מערכת החזר הוצאות

## מה צריך להכין פעם אחת

### 1. חשבון Anthropic (AI לקריאת חשבוניות)
1. כנס ל: https://console.anthropic.com
2. הירשם עם Gmail
3. לחץ על "API Keys" → "Create Key"
4. שמור את המפתח (מתחיל ב-`sk-ant-...`)
5. הוסף אשראי ($5 מספיקים לכמה חודשים)

### 2. Gmail App Password (לשליחת מיילים)
1. כנס ל-Google Account שלך
2. עבור ל: https://myaccount.google.com/security
3. הפעל "אימות דו-שלבי" אם לא מופעל
4. חפש "App Passwords" (סיסמאות אפליקציה)
5. בחר "Mail" → "Other" → כתוב "Invoice App" → לחץ "Generate"
6. שמור את הסיסמה (16 תווים)

### 3. GitHub (להעלאת הקוד)
1. כנס ל: https://github.com ← הירשם
2. לחץ "New repository" → שם: `invoice-app` → Public → Create
3. הורד Git: https://git-scm.com/download/mac
4. פתח Terminal והרץ:
```
cd ~/Desktop/invoice-app
git init
git add .
git commit -m "first commit"
git remote add origin https://github.com/YOUR_USERNAME/invoice-app.git
git push -u origin main
```

### 4. Render.com (אחסון חינמי)
1. כנס ל: https://render.com ← הירשם עם GitHub
2. לחץ "New +" → "Web Service"
3. חבר את ה-repository שיצרת
4. Render יזהה את ה-render.yaml אוטומטית
5. הוסף את המשתנים הסודיים (Environment Variables):
   - `ANTHROPIC_API_KEY` ← המפתח מסעיף 1
   - `GMAIL_APP_PASSWORD` ← הסיסמה מסעיף 2
6. לחץ "Deploy"

### סיום!
תקבל כתובת כמו: `https://invoice-app-xxxx.onrender.com`
שתף את הקישור עם כל מי שצריך להגיש חשבוניות.
